# | $\textcolor{RedOrange}{\text{Animacao Espacial}}$

Essa animação foi criada, como parte do projeto final da disciplina de **Sistemas Multimídias**, através da ferramenta **P5.js** (versão web do **Processing**) e ilustra, de forma criativa, como seria um passeio ou uma viagem espacial.



## | $\textcolor{Yellow}{\text{Projeto:}}$
 
 * [Bibliotecas.](Bibliotecas)
 * [Arquivo da projeto.](index.html)
 * [Implementação da animação.](Sketch.js)



## | $\textcolor{Lime}{\text{Classes:}}$

 * [Estrelas](Classe_Star.js):
 * Esse código foi retirado de um vídeo do [The Code Train](https://www.youtube.com/watch?v=17WoOqgXsRM) onde o desafio era implementar um "**Campo de estrelas**", ou "**Dobra espacial**", inspirado no exemplo [Starfield](https://cs.brynmawr.edu/gxk2013/examples/transformations/starfield/). A classe apresenta funcoes para desenhar e atualizar as estrelas, e foi adapdata para incluir estrelas coloridas no projeto.
 
 * [Personagem](Classe_Personagem.js):
 * Esse código foi implementado para criar um personagem humanóide. A claase fornece funções para desenhar e movimentar o personagem configurado.
